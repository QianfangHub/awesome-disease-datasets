
# [Awesome-Disease-Datasets](https://github.com/QianfangHub/awesome-disease-datasets)

> **汇聚千方病案，燃料医疗 AI。** A curated list of 200+ high-quality open-source and clinical disease datasets, clinical case sets, and multimodal medical resources.

本项目致力于打破医疗 AI 的“数据燃料荒”。我们按照 **ICD 疾病分类体系** 与 **多模态科学分类**，深度盘点、收录和编排全球范围内高质量的疾病数据集资源。

## 🤝 商业定制与临床级数据管道服务 (Enterprise Data Services)

虽然公开数据能够帮助开发者快速进行模型基线（Baseline）的验证，但在实际的**新药研发（AI 制药）、临床级医疗器械训练以及真实世界证据（RWE）研究**中，公开数据通常面临着**样本量不足、模态缺失、无随访纵向记录或伦理资质不合规**的问题。

**千方病案医数集 (QianfangHub)** 依托于深厚的医院端临床数据治理与合规脱敏管道，为企业级客户提供全栈定制服务：

1. **多模态联合对齐：** 支持“患者主诉 EMR 文本 + 连续随访病历 + 高清 DICOM 影像（CT/MRI/PET）+ 组织病理切片 + 生物标志物监测”的多维交叉对齐。
2. **专家级双盲标注：** 所有定制数据集均可指定由三甲医院副主任医师及以上级别专家进行双盲病变分割、靶点标注。
3. **100% 依法合规：** 严格遵循《数据安全法》与《个人信息保护法》，确保数据“可用不可见，入域不离域”（支持隐私计算/联邦学习接入）。
